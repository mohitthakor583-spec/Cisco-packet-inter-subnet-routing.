# Cisco-packet-inter-subnet-routing.
This project demonstrates the configuration of Inter-Subnet Routing using a Layer 3 Router and Layer 2 Switches.

## Features ##:
- Configured 2 distinct subnets: `10.10.10.0/24` and `10.10.20.0/24`.
- Implemented Default Gateways on Router interfaces (`10.10.10.1` and `10.10.20.1`).
- Resolved IP conflicts and verified end-to-end connectivity using ICMP Ping.
